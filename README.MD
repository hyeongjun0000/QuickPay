# QuickPay

**버튼 한 번으로 5억을 입금 받을 수 있는 혁신적인 간편 송금 앱입니다.** 원하는 은행을 선택하고 계좌번호를 입력한 뒤 [받기] 버튼을 누르기만 하세요.
그리고 그 이후에는... **아주 재미있는 일이 일어납니다. 🤫**

---

## 경고 및 주의 사항 (WARNING)
* **절대 공공장소(지하철, 도서관, 직장, 학교 등)나 부모님 앞에서 실행하지 마세요.**
* 본 앱을 실행하여 발생하는 친구와의 절교, 사회적 지위 하락, 스마트폰 파손(물리적 타격)에 대해 개발자는 일절 책임지지 않습니다.
* 볼륨을 미리 줄여두는 꼼수는 통하지 않습니다.

## 핵심 기능
* **완벽한 위장 UI:** 누구도 장난 앱이라 의심할 수 없는 깔끔하고 세련된 금융 앱 디자인.
* **강제 볼륨 100% (오버라이드):** 버튼을 누르는 즉시 시스템 미디어 볼륨을 최대치로 강제 상승 및 고정시킵니다.
* **무한 루프 사운드 테러:** 사용자가 당황하여 허둥대는 동안 잊을 수 없는 타격감의 사운드가 무한 반복 재생됩니다.

## 기술 스택
* **Framework:** Flutter
* **Language:** Dart
* **Core Plugins:** * `perfect_volume_control` (기기 볼륨 강제 제어)
  * `audioplayers` (오디오 무한 루프 재생)

## 설치 및 테스트 방법

플랫폼(희생양의 스마트폰 기종)에 따라 설치 및 배포 방법이 다릅니다. 악용 방지를 위해 설치 파일은 GitHub에 직접 제공하지 않으므로, 저장소를 클론한 뒤 직접 빌드해야 합니다.

### 1. 아이폰 (iOS)
애플의 강력한 보안 정책으로 인해 카카오톡 등 외부 링크로 앱을 설치할 수 없습니다. 오직 Mac과 케이블이 연결된 오프라인 상태에서만 설치가 가능합니다.
1. 타겟의 아이폰을 Mac에 케이블로 연결합니다.
2. 터미널을 열고 아래 명령어를 입력하여 앱을 단독 실행 모드로 심어줍니다.
   ```bash
   flutter run --release
```
3. 설치가 완료되면 선을 뽑고 자연스럽게 폰을 돌려줍니다.

## 🤖 2. 안드로이드 (Android) - "독사과 원격 배달"
안드로이드는 아주 자비롭습니다. 빌드된 파일(APK) 하나만 전송하면 기종 상관없이 어디서든 설치할 수 있습니다.

터미널에 아래 명령어를 입력해 설치 파일을 추출합니다.

```Bash
flutter build apk
```
빌드가 완료되면 build/app/outputs/flutter-apk/ 경로에 생성된 app-release.apk 파일을 찾습니다.

이 파일을 타겟에게 카카오톡이나 이메일로 전송한 뒤, "새로 나온 페이 앱인데 이벤트 중"이라며 설치를 유도합니다.
